使用方法

mkdir /var/frp && mkdir /var/frp/conf &&cd /var/frp/conf

建立镜像

sudo docker run --name frps -d -v /var/frp/conf:/var/frp/conf -p 5000-5100:5000-5100 -p 7000:7000 -p 7500:7500 -p 7001:7001 -p 8089:80 -p 8443:443 anlilei/frps

注意

1.默认http、https端口80和443，开放端口5000-5100TCP端口，其他需要的自行配置frps.ini。
2.修改frps.ini后docker restart frps
3.只适用于64位的系统
